# Разработка

## Начало проекта

1. Создать виртуальное окружение и активировать:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
## Работа с проектом

1. Сборка:
   ```bash
   make html
   ```

2. Запуск сервера:
   ```bash
   make run
   ```
