
=============
Что такое SSH
=============

SSH (Secure SHell — защищенная оболочка) – сетевой протокол прикладного уровня, предназначенный для безопасного удаленного доступа к UNIX-системам. Его особенность заключается в том, что вся передаваемая информация шифруется.
